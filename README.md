# CREATE FLASK 

### 1. Приложение
Данное приложение созданно мной в целях более удобного использования фреймворка Flask. 
Приложение создает базовую структуру проекта и создает пару файлов
Структура проекта выглядит так
```text
name/
├── app.py                # Основной файл приложения
├── templates/
│   └── index.html        # Главная страница
├── static/
│   ├── styles/
│   │   └── styles.css
│   └── js/
│       └── main.js       # Клиентская логика
│   └── fonts             # Шрифты
│   └── images            # Изображения
```
Из этого наполненными файлами являются токльо
```text
app.py
index.html
styles.css
```
## 2. Версии проекта
Текущая версия ```0.1```

Пока что проект находится на стадии разработки, не до конца доведена возможность создания venv для python. 
Также приложение работает с python не ниже 3.12.*! Проверено на MacOS macOS 15.6.1 24G90 arm64 и на Arch Linux.
