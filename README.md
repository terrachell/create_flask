- ## 1. Приложение
Данное приложение создано в целях более удобного использования фреймворка Flask. 
Приложение создает базовую структуру проекта и создает пару файлов

Структура создаваемого через приложение create-flask проекта выглядит так
```text
name/
├── app.py                # Основной файл приложения
├── templates/
│   └── index.html        # Главная страница
├── static/
│   ├── styles/
│   │   └── styles.css
│   └── js/
│       └── main.js       # Клиентская логика
│   └── fonts             # Шрифты
│   └── images            # Изображения
```
Из этого наполненными файлами являются только
```text
app.py
index.html
styles.css
```

- ## 3. Установка и использование
Клонируйте репозиторий и запустите installer.py
```bash
git clone https://github.com/terrachell/create_flask.git
cd create_flask
python3 installer.py
```

После чего приложение Вас оповестит об успешной установке

Далее вы можете использовать
```bash
create-flask
```

Данная команда начнет создание проекта в текущей директории

Для выбора определенной директории её нужно прописать
```bash
create-flask <name>
```

Вас попросят ввести порт и целое число 0 или 1 в зависимости от того, хотите ли вы использовать DebugMode
```bash
create-flask new
```
```text
-> Запускается сервер
-> Введите аргументы для запуска
-> Оставьте поле пустым для завершения
-> <порт <(1024-65535)> <дебагмод (0/1)>
<- 12345 1
```
И Вы получаете на выходе
```python
app.run(port=12345, debug=1)
```

Для более конкретной информации введите
```
create-flask --help
```
Вас оповестят о версии приложения, ее требования, а также где она была уже запущена и отработала штатно. Также вы увидите полный перечень аргументов доступных для использования с этим приложением.

- ## 6. Удаление

Так как create-flask устанавливается через символические ссылки, удаление исходной директории приводит к неработоспособности приложения.

Чтобы полностью удалить:

```bash
# Удалить символическую ссылку
sudo rm /usr/local/bin/create-flask

# Удалить исходную директорию
rm -rf ~/путь/к/create_flask
```
